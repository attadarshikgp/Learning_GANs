# Learning_GANs
this is a repo of projects I made in order to build my concepts up to GANs 

I found the ideas of neural networks and deep learning intriguing, but my course on the fundamentals of machine learning didn't teach me anything more than dry theory and math with no real-world application nor any understanding of the code or libraries that are industry standard (typical IIT courses). Applications of DL created new opportunities for my personal projects because of my innate interest in robotics, which, combined with my fascination with generative AI, pushed me into learning more about these concepts.

I've built my foundation in AI by diving into official PyTorch tutorials, starting with the basics and moving up to advanced projects like DCGANs (a type of AI that can generate new data). To make sure I truly understand the 'big picture,' I pair my coding with expert insights from IBM Technology and cross-reference my work with industry research papers. This approach helps me stay hands-on with the code while staying grounded in the latest technical standards.
